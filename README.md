# The Marveltonez Website v11.3 — Compliance & Corrections

Deployment-ready static website for GitHub Pages / Cloudflare.

Built from the verified live v11.2 Git commit (`fe290f3`, Website Quality Pass).

This release adds a bilingual Privacy & Datenschutz page, equal-choice cookie/media consent controls, consent-gated YouTube embeds, locally hosted fonts, the corrected Blocked & Deleted audio reference, and the approved Marveltonez Studio Hoodie imagery. It also restores the clean white-background merchandise presentation used before v11.2’s temporary charcoal thumbnails.

The consent preference is stored in first-party local storage for up to 180 days. Optional YouTube content remains blocked unless the visitor consents. Analytics is not currently used.

Deploy by uploading the **contents** of this folder to the repository root, preserving the `assets` and `metadata` directories.

After deployment, verify GitHub Pages, the live custom domain, the cookie choices, the YouTube blocked/allowed states, the Studio Hoodie page, and Blocked & Deleted playback. The official Website Version Register is maintained in the Master Planning workstream.
