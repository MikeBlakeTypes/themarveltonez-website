# The Marveltonez Website v11.1 — Catalogue Master Sync

Public-launch build for GitHub Pages / Cloudflare Pages.

Built from the v11.0 Public Launch package. All nine public song cards and expanded profiles are synchronised from Marveltonez Catalogue Master v2.1. Expanded profiles include clear close controls at the top and bottom, plus desktop click-anywhere-to-close behaviour. The catalogue allows only one song to play at a time, embedded YouTube videos start with captions off, Writer’s Collection return navigation preserves context, and the v11.0 structure, styling and product galleries remain intact.

The package references the existing `assets/images` directory from the website repository.
